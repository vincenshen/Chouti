Host Management

作者：沈洪斌
博客：http://www.cnblogs.com/vincenshen/

程序介绍：
    后端基于Django web框架
    数据库为sqlite3
	
实现功能：
	登录验证码
	
	点赞动态效果
	
	多级评论效果
	
	KindEditor插件

    用户管理：
	注册
	登录
	注销

    新闻管理：
	新建图片新闻
	查看

		
    Ajax实现：
	用户注册界面交互体验
	用户登录界面交互体验
	图片上传	
		
注意事项：
    Django版本为1.11.x
	
使用方法：
    启动服务端：pyhton manage.py  runserver
    首页地址： http://127.0.0.1:8000
    默认用户： alex 123456
